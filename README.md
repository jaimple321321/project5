# 项目说明

本项目实现了一个基于二叉搜索树的“迷宫”求解器。`BinaryTreeMaze` 会读取节点数据文件，将每行的名称和生命值载入二叉搜索树，并在生命值耗尽前寻找能够抵达最深层叶子的所有路径。路径以节点名称的顺序输出。

## 主要组件
- **BST**：通用二叉搜索树实现，负责插入、删除、查询及高度计算等基础操作。
- **Maze**：继承自 `BST<MazeNode>`，实现迷宫规则（移动先扣 1 点生命，再获取子节点生命值；生命值为负则无法继续）并记录所有可行路径。
- **MazeNode**：存储节点名称与生命值，并按名称字典序参与排序。
- **Hero**：跟踪角色当前生命值的辅助类。
- **BinaryTreeMaze**：命令行入口，读取文件、构建迷宫并打印结果。

## 仓库结构
- `BinaryTreeMaze.java`：包含 `main` 入口并处理命令行参数、文件读取及结果输出。
- `Maze.java`、`MazeNode.java`、`BST.java`：实现二叉搜索树及迷宫逻辑。
- `Hero.java`：封装生命值更新与查询。
- `README.md`：项目说明（本文档）。

## 输入格式
- 纯文本文件，每行包含两个字段：`<名称> <生命值>`，以空格分隔。
- 空行或格式不符合要求的行会被忽略。
- 树按名称字典序插入，因此输入顺序会影响最终结构，从而影响最深叶子的深度。

示例：
```
root 3
aaa 2
zzz -1
bbb 4
```

## 构建与运行
1. 在项目根目录编译所有源文件（输出到 `out` 目录）：
   ```bash
   javac -d out *.java
   ```
2. 运行程序并传入输入文件路径：
   ```bash
   java -cp out project5.BinaryTreeMaze data.txt
   ```

## 输出结果
- 如果文件缺失或参数数量不正确，程序会给出错误提示。
- 如果迷宫为空，则输出 `Maze does not contain any nodes.`
- 否则每行输出一条可行路径（从根到叶的节点名称），仅包含能够在全程保持非负生命值并抵达最深层叶子的路径。
